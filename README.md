# stylelint-brunch #

Add [Stylelint](http://www.stylelint.io) support to [brunch](http://www.brunch.io).

# Usage #

Install the plugin via npm with npm install --save-dev stylelint-brunch.

# Options #

```javascript
config = {
    plugins: {
        stylelint: {
        }
    }
}
```
